Pemrograman Dasar

Pemrograman dasar adalah fondasi utama untuk memahami bagaimana komputer dapat menyelesaikan masalah secara otomatis.

variable

Variabel dalam pemrograman adalah sebuah nama atau identifikasi yang diberikan untuk menyimpan suatu nilai data dalam memori komputer. Nilai ini dapat dapat diubah atau dimanipulasi selama eksekusi program.

array

Array adalah struktur data yang digunakan untuk menyimpan nilai dalam satu variabel. Setiap nilai dalam array dapat diakses menggunakan indeks atau posisi tertentu.

fungsi

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu. Fungsi dapat menerima input (parameter) dan menghasilkan output (nilai kembali) setelah menjalankan serangkaian instruksi.

tipe data

Tipe data mendefinisikan jenis nilai yang bisa disimpan oleh variabel. Contoh umumnya meliputi String (teks), Integer (angka bulat), Float (desimal), dan Boolean (true/false)

percabangan

Percabangan (If-Else) memungkinkan program mengambil keputusan berdasarkan kondisi tertentu. Jika kondisi terpenuhi, maka perintah A akan diajalnkan; jika tidak, jalankan perintah B.

perulangan

Perulangan (Loop) digunakan untuk mengeksekusi blok kode yang sama berkali-kali secara otomatis selama kondisi yang ditentukan masih terpenuhi (seperti For atau While)
