01

Menjembatani Manusia dan Mesin: Algoritma

Bahasa yang kita gunakan sehari-hari penuh dengan ambiguitas dan konteks. Ketika kita berkata "tolong buatkan kopi", manusia lain mengerti apa yang harus dilakukan berdasarkan intuisi dan pengalamannya.

Namun, komputer tidak memiliki intuisi. Ia adalah benda mati yang hanya memahami dua hal: ada arus listrik (1) dan tidak ada arus listrik (0). Agar "bahasa manusia" tadi bisa dimengerti oleh "logika komputer", kita harus membangun sebuah jembatan.

Jembatan itulah yang disebut dengan Algoritma. Algoritma adalah urutan langkah-langkah logis, pasti, dan tidak ambigu yang dirancang untuk memecahkan sebuah masalah. Komputer membutuhkan algoritma dengan presisi absolut:

  1. Verifikasi apakah air di dalam teko lebih dari 250ml. Jika tidak, isi air.
  2. Nyalakan elemen pemanas teko.
  3. Pantau sensor suhu hingga mencapai tepat 95 derajat Celcius.
  4. Tuangkan 15 gram bubuk kopi ke dalam gelas.
  5. Tuangkan air panas ke dalam gelas perlahan.
"Bahasa pemrograman hanyalah kamus penerjemah. Kemampuan menyusun logika algoritmalah yang menjadi inti dari keahlian seorang programmer sejati."
02

Variabel dan Manipulasi Memori

Sebelum komputer bisa memproses algoritma, ia harus "mengingat" data apa yang sedang ia kerjakan. Di sinilah peran memori (RAM) dan konsep Variabel menjadi sangat krusial.

Secara fisik, memori komputer terdiri dari miliaran sel kecil. Membaca dan menulis di alamat fisik memori tersebut (misalnya: 0x7FFF5FB) sangatlah rumit. Oleh karena itu, kita menggunakan Variabel.

Variabel adalah "alias" atau label yang kita tempelkan pada sebuah laci di dalam memori. Alih-alih mengingat kode mesin yang rumit, kita cukup memberi nama laci tersebut dan mendeklarasikan Tipe Data apa yang akan masuk ke laci tersebut (Teks, Angka Bulat, Angka Desimal, atau Logika).

Struktur Alokasi Memori di C++
#include <iostream>
#include <string>
using namespace std;

int main() {
    // Program mengalokasikan RAM sesuai dengan wujud data yang masuk
    string tipeTeks = "Halo Dunia";
    int tipeAngkaBulat = 100;
    float tipeAngkaPecahan = 3.14;
    bool tipeLogika = true; // Logika Biner (1 atau 0)

    cout << "Data berhasil disimpan di dalam laci RAM!";
    return 0;
}

03

Operator dan Komputasi Mesin

Data yang diam di dalam memori tidak akan menghasilkan program yang berguna. Kita harus memanipulasinya menggunakan ALU (Arithmetic Logic Unit)—bagian dari prosesor yang bertugas melakukan perhitungan matematika.

Dalam C++, kita menginstruksikan prosesor dengan menggunakan simbol-simbol khusus yang disebut Operator. Berikut adalah kategori utamanya:

  • Operator Aritmatika: Layaknya kalkulator (+, -, *, /). Ada satu operator krusial yaitu Modulo (%) yang berfungsi mencari sisa hasil bagi. (Contoh: 10 dibagi 3 sisanya 1).
  • Operator Perbandingan: Membandingkan dua nilai (>, <, ==) dan selalu mengembalikan nilai Boolean logika (true/false).
  • Operator Logika: Digunakan untuk menggabungkan syarat. AND (&&) mewajibkan semua syarat benar, sedangkan OR (||) cukup salah satu saja yang benar.
Aritmatika & Perbandingan di C++
#include <iostream>
using namespace std;

int main() {
    int num1 = 10;
    int num2 = 3;

    // 1. Melakukan komputasi Aritmatika (Sisa Bagi / Modulo)
    int hasilBagi = num1 % num2;

    // 2. Melakukan evaluasi Perbandingan
    bool cekLebihBesar = (num1 > num2);

    cout << "Sisa bagi (Modulo)  : " << hasilBagi << endl;
    cout << "Apakah 10 > 3?      : " << cekLebihBesar; // Muncul 1 (True)
    
    return 0;
}

Percabangan (Control Flow)

Aplikasi modern tidak hanya berjalan lurus. Ia bisa "berpikir" dan merespons situasi secara berbeda, layaknya sensor pintar yang bereaksi terhadap lingkungan. Kemampuan memilih rute ini disebut Percabangan.

Dalam C++, logika ini dibangun menggunakan struktur if (jika kondisi benar), else if (atau jika kondisi lain yang benar), dan else (bila semua kondisi salah). Penting diingat: begitu komputer menemukan satu blok yang bernilai true, ia akan mengeksekusi blok tersebut lalu langsung melompat keluar, mengabaikan semua syarat di bawahnya.

Arsitektur Percabangan di C++
#include <iostream>
using namespace std;

int main() {
    int suhu = 25;

    // Program akan mengecek satu per satu dari atas ke bawah
    if (suhu <= 0) {
        cout << "Air berwujud Padat (Es)";
    } 
    else if (suhu >= 100) {
        cout << "Air berwujud Gas (Uap)";
    } 
    else {
        // Kondisi ini yang tereksekusi!
        cout << "Air berwujud Cair";
    }
    
    return 0;
}

05

Perulangan Logika (Looping)

Batas terdalam ketahanan manusia adalah kelelahan fisik. Sebaliknya, keunggulan mutlak mesin adalah kemampuannya menjalankan instruksi yang sama jutaan kali secara akurat tanpa merasa bosan.

Di dunia pemrograman, kita sangat menghindari *copy-paste* kode yang sama. Kita menggunakan Looping. Jenis perulangan yang paling sering dipakai adalah For Loop. Ia memerlukan 3 parameter utama di dalam kurungnya:

  1. Start: Dari angka berapa perhitungan dimulai.
  2. Stop: Sampai angka berapa perulangan ini harus terus berputar.
  3. Step: Modifikasi nilai setiap putaran selesai. (Simbol i++ berarti nilai variabel i ditambah 1).
Struktur For Loop di C++
#include <iostream>
using namespace std;

int main() {
    int batas = 5;

    // Format: for (Start ; Stop ; Step)
    for (int i = 1; i <= batas; i++) {
        // Perintah ini akan dipaksa berjalan persis 5 kali
        cout << "Render grafis siklus ke-" << i << endl;
    }
    
    return 0;
}

Fungsi: Membangun Mesin Modular

Ketika kodemu mencapai ribuan baris, menuliskan instruksi yang lurus terus menerus akan menjadikannya berantakan dan mustahil diperbaiki. Solusinya adalah memecahnya menjadi modul-modul logika kecil yang disebut Fungsi.

Bayangkan Fungsi seperti sebuah "Mesin Pabrik Pengolah". Kamu cukup merakitnya satu kali, dan memanggil/menekan tombolnya berkali-kali saat dibutuhkan. Mesin ini bisa menerima Parameter (bahan baku input), mengolahnya, dan menghasilkan Return (produk output akhir).

Deklarasi & Pemanggilan Fungsi di C++
#include <iostream>
#include <string>
using namespace std;

// 1. TAHAP MERAKIT MESIN PENGOLAHAN (Deklarasi)
string buatProduk(string bahan) {
    string hasil = "Log: Menerima " + bahan + ". Output berhasil dibuat!";
    return hasil; // Mengembalikan (Return) hasil akhir mesin
}

int main() {
    // 2. TAHAP MENGGUNAKAN MESIN (Pemanggilan Parameter)
    string outputAkhir = buatProduk("Daun Teh Hijau");
    
    cout << outputAkhir;
    return 0;
}
